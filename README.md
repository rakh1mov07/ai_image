\# AI Camera Editor



AI Camera Editor — это интерактивный редактор изображений и камеры на Python с поддержкой искусственного интеллекта через LM Studio.



Программа позволяет работать как с веб-камерой в реальном времени, так и с готовыми изображениями. Пользователь может применять эффекты вручную через меню или описывать их обычным текстом, а локальная языковая модель автоматически преобразует команды в операции обработки изображений.



\---



\## Возможности



\### Работа с камерой



\* Просмотр изображения с веб-камеры в реальном времени;

\* Применение эффектов к видеопотоку;

\* Сохранение текущего кадра;

\* Отмена последнего действия;

\* Сброс всех эффектов.



\### Работа с фотографиями



\* Загрузка изображений через диалоговое окно;

\* Загрузка изображения по указанному пути;

\* Поддержка форматов:



&#x20; \* JPG / JPEG

&#x20; \* PNG

&#x20; \* BMP

&#x20; \* TIFF

&#x20; \* WEBP



\### Поддержка искусственного интеллекта



Интеграция с LM Studio позволяет использовать естественный язык:



Примеры команд:



```

сделай изображение чёрно-белым

поверни фото на 90 градусов

увеличь яркость

добавь размытие

сделай эффект сепии

выдели красный канал

```



Модель автоматически определяет нужную операцию и применяет её.

\## Поддерживаемые эффекты



| №  | Эффект                     |

| -- | -------------------------- |

| 1  | Поворот на 90°             |

| 2  | Отражение по горизонтали   |

| 3  | Чёрно-белое изображение    |

| 4  | Красный канал              |

| 5  | Зелёный канал              |

| 6  | Синий канал                |

| 7  | Увеличение яркости         |

| 8  | Уменьшение до 50%          |

| 9  | Размытие                   |

| 0  | Сброс всех эффектов        |



Дополнительно поддерживаются:



\* Бинаризация;

\* Произвольная обрезка изображения;

\* Масштабирование;

\* Последовательное применение нескольких эффектов.

\## Горячие клавиши



| Клавиша  | Действие                  |

| -------- | ------------------------- |

| ESC      | Выход из программы        |

| Ctrl + S | Сохранить кадр            |

| TAB      | Сбросить все эффекты      |

| ←        | Отменить последний эффект |

| O        | Открыть фотографию        |

| K        | Обрезать изображение      |

| Z        | Увеличить изображение ×2  |

| C        | Вернуться к режиму камеры |

| 1–9      | Быстрый вызов эффектов    |

\## Установка



\### 1. Клонирование проекта



```bash

git clone https://github.com/USERNAME/ai-camera-editor.git

cd ai-camera-editor

```



\### 2. Установка зависимостей



```bash

pip install opencv-python numpy requests pillow

```



При необходимости установить tkinter:



\#### Ubuntu/Debian



```bash

sudo apt install python3-tk

```



\---



\## Настройка LM Studio



1\. Установите LM Studio.

2\. Загрузите модель:



```

Llama 3.1 8B

```



3\. Включите локальный сервер OpenAI API.

4\. Убедитесь, что сервер доступен по адресу:



```python

http://localhost:1234/v1/chat/completions

```



При необходимости измените настройки в коде:



```python

LM\_STUDIO\_URL = "http://localhost:1234/v1/chat/completions"

MODEL\_NAME = "llama-3.1-8b"

```



\---



\## Запуск



```bash

python main.py

```



После запуска откроется окно камеры и консольное меню.



\---



\## Использование



\### Команды через меню



Введите номер операции:



```text

>>> 3

```



Результат:



```

Чёрно-белый эффект применён

```



\### Команды через ИИ



Введите текст:



```text

>>> сделай фото более ярким

>>> добавь размытие

>>> выдели контуры

```



Модель автоматически преобразует запрос в соответствующую операцию.



\---



\## Обрезка изображения



1\. Нажмите `K`;

2\. Выделите область мышью;

3\. Нажмите `Enter` для применения;

4\. Нажмите `Esc` для отмены.
