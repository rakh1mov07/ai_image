#  AI Camera Editor

Редактор видео с камеры и фотографий в реальном времени с управлением через **искусственный интеллект** (LM Studio + llama 3.1 8B) и **OpenCV**.

Просто напиши в терминале что хочешь сделать с изображением — ИИ сам поймёт команду и применит нужный эффект.

---

## Возможности

-  **Режим камеры** — обработка видеопотока в реальном времени
-  **Режим фото** — загрузка изображения с диска и редактирование
-  **ИИ-управление** — вводи команды на русском языке, ИИ сам выбирает эффект
-  **Интерактивная обрезка** — выдели область мышью прямо в окне
-  **Отмена** — отменяй эффекты по одному стрелкой влево
- 16 встроенных эффектов с быстрым вызовом по цифрам

---

##  Требования

- Python 3.10+
- [LM Studio](https://lmstudio.ai/) с загруженной моделью **llama-3.1-8b**

##  Установка

```bash
pip install opencv-python numpy requests pillow
```

> `tkinter` входит в стандартную библиотеку Python. Если отсутствует (Linux):
> ```bash
> sudo apt install python3-tk
> ```

##  Запуск

1. Открой **LM Studio**, загрузи модель `llama-3.1-8b`
2. Перейди в **Local Server → Start Server** (порт `1234`)
3. Запусти скрипт:

```bash
python ai_camera_editor.py
```

##  Управление

### Горячие клавиши (в окне камеры)

| Клавиша | Действие |
|---|---|
| `ESC` | Выход |
| `Ctrl+S` | Сохранить кадр (`snapshot_NNNN.png`) |
| `TAB` | Сбросить все эффекты |
| `←` (стрелка влево) | Отменить последний эффект |
| `O` | Открыть фото с диска (диалог) |
| `K` | Обрезать текущий кадр/фото мышью |
| `Z` | Увеличить ×2 |
| `C` | Вернуться к камере |
| `1` – `9` | Быстрый вызов эффекта из меню |

### Команды в терминале

| Команда | Действие |
|---|---|
| `1` – `16` | Применить эффект по номеру |
| `photo` | Открыть диалог выбора фото |
| `photo C:\путь\к\файлу.jpg` | Загрузить фото напрямую по пути |
| `crop` | Запустить инструмент обрезки |
| `camera` | Вернуться к режиму камеры |
| `0` / `reset` | Сбросить все эффекты |
| Любой текст | Отправить команду в ИИ |

### ИИ-команды (примеры)

```
>>> сделай чёрно-белым
>>> добавь немного размытия
>>> повысь яркость
>>> переверни изображение
>>> сделай негатив
```

##  Эффекты

| № | Эффект |
|---|---|
| 1 | Поворот на 90° |
| 2 | Отражение по горизонтали (зеркало) |
| 3 | Чёрно-белое |
| 4 | Красный канал |
| 5 | Зелёный канал |
| 6 | Синий канал |
| 7 | Увеличение яркости |
| 8 | Уменьшение до 50% |
| 9 | Размытие (Gaussian Blur) |
| 10 | Инверсия цветов |
| 11 | Повышение контраста |
| 12 | Сепия |
| 13 | Контуры (Canny Edge) |
| 14 | Резкость |
| 15 | Тиснение (Emboss) |
| 16 | Увеличение до 200% |
| 0 | Сброс всех эффектов |

Эффекты **накапливаются** — можно применять несколько подряд. Отменять по одному — стрелкой `←`, сбросить все — `TAB`.

##  Инструмент обрезки

Нажми `K` в окне или введи `crop` в терминале:

1. Откроется отдельное окно с текущим кадром
2. Зажми **левую кнопку мыши** и выдели нужную область
3. В углу отображается размер выделения в пикселях
4. **Enter** — применить обрезку
5. **ESC** — отмена

> Если нажать `K` в режиме камеры — кадр автоматически замораживается перед обрезкой.

##  Настройки

В начале файла `ai_camera_editor.py` можно изменить:

```python
LM_STUDIO_URL = "http://localhost:1234/v1/chat/completions"  # адрес LM Studio
MODEL_NAME    = "llama-3.1-8b"   # модель
CAMERA_INDEX  = 0                 # индекс камеры (попробуй 1 если не открывается)
FONT_PATH_OVERRIDE = ""           # путь к своему TTF-шрифту (опционально)
```
